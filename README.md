# Jotte
Jotte is a simple note taking tool built in [Compose Multiplatform](https://kotlinlang.org/compose-multiplatform/).
It's designed to support android 🤖 and ios 🍎 build targets
and contains features such as
* Image Capture 📸
* Audio Recording 🎤
* Links 🔗
* Whiteboard ✍️
* Metrics 📈
* Fun sound effects 🎶

Roadmap Features include ~
* Video Capture
* Transformable text in whiteboard (Drag, Scale, Rotate)
* Media Uploading (files)
* PDF exports

This project was built simply to better my understanding of Compose Multiplatform and to enjoy the benefits of X-platform development!
